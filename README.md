## Getting a Clone
Once Git is installed to have a local clone of the repository:

```sh
mkdir C:\DevOpsMC
cd c:\DevOpsMC
git clone https://github.com/johnthebrit/DevOpsMC.git .
```

To update, make sure you are in the folder downloaded to and run

```sh
git pull
```
